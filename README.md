# üìù Blogging Website (Full-Stack MERN)

A full-stack blogging platform where users can create, read, update, and manage blogs.
This project is built to simulate a **real-world production application**, focusing on authentication, role-based access, and deployment.

---

## üöÄ Features

- User Authentication (Register / Login)
- JWT-based Authorization
- Create, Read, Update, Delete (CRUD) Blogs
- Admin & User Role Management
- Profile Update Functionality
- Secure REST APIs
- Image upload support
- Responsive User Interface
- Production-level Deployment

### Frontend
- React / Next.js
- JavaScript (ES6+)
- Axios
- CSS / Tailwind CSS

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Cloudinary (for image handling)

- Backend: Render
